# Lern-Periode 8

## Godot
Ich habe mich für Godot entschieden, weil ich mich allgemein für Spiele interessiere und auch in meiner Freizeit 
gerne ein eigenes Spiel programmieren möchte.
Godot bietet mir eine gute Möglichkeit, genau das umzusetzen, was mich interessiert.
Ich sehe diese Entscheidung als Chance, praktische Erfahrungen zu sammeln und Schritt für Schritt zu lernen, 
wie Spieleentwicklung funktioniert. Ich freue mich darauf, herauszufinden, wie die einzelnen Teile zusammenarbeiten, und am Ende zu sehen, wie mein eigenes Spiel aussehen wird.
